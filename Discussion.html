The paper

  • Can we infer election fraud from altered tallies using machine learning?
  • 53,000 images of vote tally sheets at the polling station level
  • Image recognition via convolutional neural networks to identify tallys that have been altered
  • training set of 900, validation set of 125

Start with simple image recognition

  • Each image is represented in W x H X RBG numbers, where RBG areranges from 0 (black) to 255 (white)
  • Here: 227 * 227 * 3

Start simple

  • Easiest way to classify, would be to compare each pixel
  • E.g. L1 distance \[d_{1}(I_{1}, I_{2}) = \Sigma_{p}\mid I_{1}^{p} - I_{2}^{p}\mid\] or L2 (euclidian) \[d_{2}(I_{1}, I_{2}) = \Sigma_{p} \sqrt{ ( I_{1}^{p} - I_{2}^{p})}\]
